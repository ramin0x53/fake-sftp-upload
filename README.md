# fake-sftp-upload
## Usage:
```
  -c int
        count (default 10)
  -dp string
        destination path (default "/tmp")
  -p string
        remote port (default "22")
  -pa string
        password
  -r string
        remote host
  -s string
        source file
  -t string
        times (default "04:00")
  -u string
        username
```
